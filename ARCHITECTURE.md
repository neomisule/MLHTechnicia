```http
POST http://localhost:8000/clone-voice
Content-Type: multipart/form-data

user_id: user_1234567890
voice_name: MyVoice
audio_file: recording.mp3
```

### 2. Clone Voice Response
```json
{
  "success": true,
  "voice_id": "abc123xyz789",
  "message": "Voice cloned successfully!"
}
```

### 3. Full Interaction Request
```http
POST http://localhost:8000/full-interaction
Content-Type: multipart/form-data

user_id: user_1234567890
voice_id: abc123xyz789
audio_file: question.mp3
```

```
Headers:
  X-Transcript: "What is artificial intelligence?"

  [Audio MP3 data in user's cloned voice]
```
